# The bin/ toolbelt

The first mate drives these; interactive entrypoints work by hand too, while `*-lib.sh` files are sourced helpers.
Each file also starts with a short header comment.
If you have changed away from the firstmate home in an interactive shell, invoke these scripts by absolute path through the repo's `bin/` directory; the scripts self-locate internally after they start.

| Script                      | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |

| `fm-session-start.sh`       | The one command AGENTS.md sections 3 and 5 run at every session start: composes `fm-lock.sh`, `fm-bootstrap.sh` (its three mutating sweeps gated on holding the lock via `FM_BOOTSTRAP_DETECT_ONLY`), and `fm-wake-drain.sh`, then prints a full context digest (`data/projects.md`, `data/secondmates.md`, `data/captain.md`, `data/learnings.md`, each `ABSENT`-marked when missing) and fleet-state digest (`data/backlog.md`, every `state/*.meta`, a bounded `state/*.status` tail, `state/.afk`, `state/.fleet-freeze` in its own `FLEET FREEZE` subsection, and a cheap per-task endpoint-liveness read); prints a loud read-only banner and skips every mutating step when the lock is held elsewhere; never arms the watcher itself                                                               |

| `fm-fleet-sync.sh`          | Fetch all clones, or one clone selected by absolute path, relative path, bare project name, or `projects/<name>` resolved against this home's projects dir; fast-forward safe default-branch states, self-heal clean detached ancestor drift, report unsafe drift as `STUCK:`, and safely prune branches whose remote is gone                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |

| `fm-brief.sh`               | Scaffold a ship brief with a worktree-isolation assertion, a report-only scout brief with `--scout`, or a secondmate charter with `--secondmate`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |

| `fm-freeze.sh`              | Toggle or report the local, gitignored `state/.fleet-freeze` incident-pause flag with `on [reason...]`, `off`, or `status`; while frozen, `fm-spawn.sh`, `fm-send.sh`, `fm-watch.sh`, `fm-watch-arm.sh`, and daemon injection all refuse unless `FM_FLEET_FREEZE_BYPASS=1` is set for that one command; does not inspect, steer, or tear down any crewmate itself                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `fm-freeze-lib.sh`          | Shared fleet-freeze guard sourced by `fm-spawn.sh`, `fm-send.sh`, `fm-watch.sh`, `fm-watch-arm.sh`, and `fm-supervise-daemon.sh`; `fm_fleet_freeze_refuse` prints the recorded reason and refuses to wake, spawn, or steer while `state/.fleet-freeze` exists unless bypassed                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |

| `fm-fleet-map-lib.sh`       | Shared tracked-state/live-endpoint reconciliation helpers sourced by `fm-fleet-map.sh` and `fm-reconcile-stale.sh`; reads `state/*.meta` into a tracked table, reads Herdr agents (or a `FM_FLEET_MAP_HERDR_JSON` fixture), and writes the stale/mismatch warnings both tools report                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| `fm-landed-work-lib.sh`     | Shared landed-work assessor deliberately more conservative than `fm-teardown.sh` (omits its PR-head patch-id containment fallback, so use `fm-teardown.sh` directly for a squash-merge case it would recognize as landed but this reports unlanded/blocked); sourced by `fm-reconcile-stale.sh`; carve-outs mirror `fm-teardown.sh`: always blocks `kind=secondmate` toward `fm-teardown.sh` and gates `kind=scout` on the report's existence alone; reports a default-branch fetch/lookup failure as `blocked` (inconclusive), never as a confirmed `unlanded` verdict                                                                                                                                                                                                                                    |
| `fm-reconcile-stale.sh`     | Dry-run-first stale tracked-state reconciler: reports every stale-tracked record from `fm-fleet-map-lib.sh` with its landed-work assessment and any operator-untracked Herdr agents, writing nothing by default; `--clean <id> --yes` re-verifies the endpoint is dead, the work path has no unlanded work, and the fleet is not frozen, then removes only that id's volatile state files and `tasktmp`                                                                                                                                                                                                                                                                                                                                                                                                    |
| `fm-home-seed.sh`           | Lease/provision a secondmate home transactionally, clone projects, initialize gates, and maintain `data/secondmates.md`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
